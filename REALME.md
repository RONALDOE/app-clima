# 🌦️ Proyecto: Landing Page y App del Clima

**Integrantes:** 
Yansell Rafael Rosario Siri

## 🚀 Funcionalidades
* Landing page de una sola sección con navegación inteligente.
* Scroll suave implementado con `scrollIntoView()`.
* Búsqueda de clima en tiempo real de cualquier ciudad.
* Muestra temperatura, descripción y un icono representativo.

## 🛠️ Tecnologías Usadas
* **HTML5:** Estructura semántica con IDs únicos.
* **CSS3:** Diseño Mobile First, Flexbox, Grid y Menú Sticky.
* **JavaScript Moderno:** `const`/`let`, Arrow Functions, Fetch API (OpenWeather) y manipulación dinámica del DOM.

## 📂 Cómo abrir el proyecto
1. Descarga o clona este repositorio en tu computadora.
2. Abre el archivo `index.html` en cualquier navegador web.
3. *Nota para evaluadores:* Para que la búsqueda funcione, se debe colocar una API Key válida de OpenWeather en el archivo `/js/app.js`.
